### ROT.js

 * [ROT.js Roguelike Toolkit in JavaScript](http://ondras.github.io/rot.js/hp/)

Use it in your HTML like this:

```html
<script src="https://cdn.jsdelivr.net/npm/rot-js@2/dist/rot.js"></script>
```
